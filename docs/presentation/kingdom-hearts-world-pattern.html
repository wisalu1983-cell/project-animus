一个入口，打开多个由光暗主线串联的主题世界。

《王国之心》的多世界结构不是单纯 IP 串场。它把 Disney / Pixar 世界做成可进入、可并肩作战、并会接入主线冲突的章节空间。

拆开看：Heartless 变体与索拉变体Shared Rule, Separate Displays

同一种世界规则会分别作用到两类外来元素：Heartless 被重画成当地敌人，索拉被重写成当地居民/行动者。两类变化要分开看，才看得清 KH 如何把 IP 世界从背景变成规则。

Track A / Sora world forms

同一个主角，进入世界后先被改写

这里单独看索拉：KH 的强点是让玩家 avatar 也服从世界规则。海底、怪诞城镇、动物生态、玩具比例、怪物社会都直接改写索拉的身体与身份。

小美人鱼

海底形态

索拉变成海中形态，玩家先获得“能在水下行动”的身体规则。

圣诞夜惊魂

万圣节形态

索拉被改成怪诞城镇的视觉风格，外来者先被本地审美同化。

狮子王

幼狮形态

索拉变成四足动物，移动方式、身体比例和生态身份都被重写。

玩具总动员

玩具形态

索拉变成玩具，儿童房和商场因此被重新理解为冒险空间。

怪兽电力公司

怪物形态

索拉以怪物身份进入工厂，先成为当地居民，再参与当地危机。

Track B / Heartless localization

同一类黑暗敌人，被不同 IP 世界重画

这里单独看 Heartless / darkness：敌人仍服务同一条主线，但外形、材质、武器和行动气质会被每个世界本地化。

小美人鱼 / 海底语法

Sea Neon 像水母

Heartless 变成水下生物轮廓，颜色、漂浮姿态和攻击方式都贴近 Atlantica 的海底环境。

圣诞夜惊魂 / 怪诞语法

Wight Knight 像万圣节怪物

敌人被改成瘦长、扭曲、怪诞的剪影，进入 Halloween Town 的惊悚喜剧视觉系统。

狮子王 / 动物生态

Living Bone 像骨架野兽

Heartless 变成动物型骨架敌人，和 Pride Lands 的草原生态、兽群冲突、动物身体规则接在一起。

玩具总动员 / 玩具语法

Toy Trooper 像玩具兵

Heartless 被设计成 action figure 式敌人，带玩具枪、喷射背包和玩具士兵轮廓。

冰雪奇缘 / 冰雪语法

Winterhorn 像冰角驯鹿

Heartless 被改成带冰晶鹿角的形态，接入 Arendelle 的雪山、冰属性和恐惧主题。

02 / 单个世界范式

先看共性矩阵，再拆一个世界如何落地

Overview Matrix / 五个世界共用的结构
Pixar / Toy Story

Toy Box

主题包玩具比例、Andy 房间、Galaxy Toys 商场、可驾驶玩具机甲。
当地角色Woody、Buzz 等玩具判断索拉一行是否可信，并与其同行。
主线接入Heartless 与 Organization 介入“玩具是否有心”的问题。
设计价值把 IP 的核心命题“玩具有生命”转译成 KH 的“心”的规则。
Pixar / Monsters, Inc.

Monstropolis

主题包怪物公司、门仓、工厂管线、惊吓/欢笑能源。
当地角色Mike、Sulley、Boo 让玩家进入“保护孩子”的本地任务。
主线接入Vanitas / Unversed 等黑暗侧力量介入本地能源与恐惧主题。
设计价值世界规则先改写主角外形，再让玩家理解当地社会秩序。
Disney / Frozen

Arendelle

主题包雪山、冰宫、暴风雪、Elsa 的冰雪力量。
当地角色Anna、Elsa、Olaf 等角色承担本地情感轴。
主线接入黑暗阵营观察 Elsa 的力量，把本地角色变成主线判断对象。
设计价值IP 的情绪主题“恐惧与自我接纳”可直接接上光暗主题。
Disney / Pirates

The Caribbean

主题包海盗船、海战、岛屿探索、宝藏与诅咒。
当地角色Jack Sparrow 等角色把玩家带入电影式冒险节奏。
主线接入Organization 追踪箱子与黑暗线索，局部冒险转成系列追踪任务。
设计价值把电影动作场面转译成航海、船战、开放海域探索。
Disney / Hercules

Olympus

主题包希腊神殿、竞技场、英雄试炼、冥界威胁。
当地角色Hercules 是英雄样板，承接索拉对“力量”的理解。
主线接入Hades 与黑暗势力让本地神话成为主线冲突的入口。
设计价值用熟悉神话空间解释主角成长，而不是只做关卡皮肤。

Toy Box 详细拆解

Layer 01 / IP 主题包

先提取“玩具尺度”，再把日常空间改成关卡

设计手法是先抓 Toy Story 最可玩的规则核：玩家和角色都处在玩具尺度。于是儿童房、商场、货架、玩具陈列不再只是电影场景，而会变成高低差、路径、遮挡物、遭遇点和可驾驶玩具机甲的来源。

Layer 02 / 当地角色参与

IP 角色不是背景，而是玩家进入 Toy Box 的关系接口

Woody、Buzz、Hamm、Rex 等角色承担三件事：在 Andy 房间接住外来者，说明“Andy 和伙伴失踪”的本地危机，再和索拉一起行动、战斗和触发 Toy Story 风格的玩法演出。
