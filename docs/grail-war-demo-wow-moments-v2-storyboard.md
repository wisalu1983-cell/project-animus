## 0. 结构总览

**一句话**：同一场圣杯战争的同一个夜晚，两对 Master-英灵搭档从各自的第一人称走向同一个战场；你（主视角）经历召唤、见证英灵的意志、最后与他并肩赢下遭遇战——输掉的对面，也是一对真人搭档。

**叙事结构**：「第一夜」连续单线 + 双第一人称（主次之分）。

- **主线 A（你）**：召唤 → 英灵擅自出巡 → 收到战报赶到战场 → 与英灵配合决胜 → 获得认可。承载全部情感弧线：敬畏 → 震动 → 被需要 → 并肩 → 认可
- **副线 B（对面玩家）**：切入 4 次、每次一帧、统一视觉标识。作用：证明"对面是真人玩家"、承载英灵差异化、提供搭档关系的"未来时"对照
- **三个 wow moment 的因果链**：圣遗物决定了英灵是谁 → 英灵的意志撑起战场 → 决定胜负的是你和他的配合。严格对应 tagline 游戏循环（召唤 → 羁绊 → 配合 → 赢）
- **Wow 2 → Wow 3 的转折点 = "你到场"**：此前你是英灵意志的旁观者，此后你是配合的参与者

**时长/篇幅配比**：Wow 1 四帧（F1-F4）/ Wow 2 四帧（F5-F8）/ Wow 3 五帧（F9-F13）。

## 1. 通用视觉语言

- **双视角标识**：主线 A 帧 = 全画幅、夜色暖金基调；副线 B 帧 = **冷蓝灰基调 + 上下黑边（letterbox）**，观众一眼识别"这是另一位玩家的视角"。标识从 F4 建立后全程一致
- **双方玩家均不露正脸**：只用手部、背影、局部。玩家是观众的化身，不是剧中角色（继承旧决策，服务第一人称代入）
- **玩家反应克制**：惊愕不夸张、胜利不欢呼、失败不崩溃（继承旧红线）
- **生成过程可见**：召唤帧（F2）必须呈现"由虚到实的中间态"而非完成态——观众要看到"正在生成"，服务 G3 因果归因
- **性格用动作不用独白**：英灵的人格通过第一个动作、战报的措辞风格、战法差异传达，不写内心独白
- **画风建议**：写实电影感概念图——Fate/Zero 式成人向冷冽夜色 + 电影打光，避免明亮卡渲二次元。最终风格与制图工具（AI 生成 / 手绘）交下游确定

## 2. 示范角色与两对搭档对照

| | 主线 A × 英灵 X | 副线 B × 英灵 Y |
|---|---|---|
| 示范英灵 | **Iskandar 型王者**（自主意志全系列最强、Waver 线还原度最高、已有项目素材） | **枪兵型佣兵**（第一夜遭遇战原作原型、与王者型差异最大化；不锁定具体原作角色） |
| 圣遗物（示范） | 一本泛黄的旧书（亚历山大远征史老版本） | 一枚变形的旧弹壳 / 一柄旧军刺 |
| 关系阶段 | 今夜刚缔约（生涩 → 信任） | 已合作多日（老练默契） |
| 战报风格 | 大开大合，称呼"小子" | 军用简报体，只说事实 |
| 配合语言 | 英灵主导出方案，Master 补关键位 | Master 主导指挥，英灵精确执行 |

B × Y 的"老练默契"是主线关系的**未来时**——观众看完自然理解"羁绊是养成出来的"，无需时间跳跃。

## 3. 帧清单（13 帧）

### Wow 1 · 圣遗物生成英灵（F1-F4）

**F1 · 深夜 · 圣遗物**（主 A · 第一人称）
- **画面**：深夜书房，第一人称俯视：双手捧着一本泛黄的旧书（封皮磨损、书脊开裂），桌角一段粉笔法阵的弧线延伸向地板暗处。暖色台灯是唯一光源
- **配文**：*你的圣遗物：一件真正属于你的旧物。它的来历，决定了谁会回应你的召唤。*
- **原作锚点**：E1 催化剂召唤——Waver 用偷来的斗篷碎片召唤出 Iskandar；催化剂的深层关联比召唤者资质更决定召唤结果
- **约束**：物品普通而有来历；此帧不发光不浮起，魔性视觉从 F2 开始；手部特写不露脸

**F2 · 召唤 · 显形**（主 A · 第一人称）
- **画面**：第一人称仰视：法阵全亮，光柱中一个魁梧身影**正在成形**——轮廓由虚到实、细节逐层浮现（披风 → 铠甲 → 面容），房间墙面被光推远变形
- **配文**：*不是从卡池里抽出，不是从列表里选择——是从你的圣遗物里，当场生成一个独一无二的人格。*
- **原作锚点**：召唤仪式的咏唱与显形；英灵应召而来时人格完整
- **约束**："由虚到实的中间态"是画面主体，不是结果直接出现（G3 归因）

**F3 · 第一句话**（主 A · 第一人称）
- **画面**：中近景：王者英灵俯身直视镜头（= 你），一手已经抄起你桌上的地球仪，咧嘴而笑；他的体量让整个房间显得狭小
- **配文**：*他睁眼的第一件事，是审视你配不配当他的 Master。第二件事，是抓起你桌上的地球仪——他有自己想要的东西。*
- **原作锚点**：E4 登场问句仪式（「問おう。貴方が私のマスターか」）；Fate/Zero 里 Iskandar 召唤当夜就沉迷世界地图——"征服世界从了解世界开始"；登场即完整人格
- **约束**：性格用动作传达；观众要体会 Waver 式"被评估"的压迫与敬畏

**F4 · 同一夜 · 另一位玩家**（副 B · 第一人称 · 冷调标识首次出现）
- **画面**：冷蓝灰基调 + 黑边。另一间公寓：桌上圣遗物是一枚变形的旧弹壳；法阵光中，清瘦的枪兵型英灵单膝跪地、垂首持枪，沉默
- **配文**：*同一个夜晚，这座城市里的另一位玩家。不同的圣遗物，召唤出完全不同的"人"。*
- **原作锚点**：多对 Master-Servant 同夜缔约的开战之夜（S2 圣杯战争框架）；催化剂差异 → 英灵差异
- **约束**：一帧完成对照，不展开；副线视觉标识从此帧建立

### Wow 2 · 英灵战场自主意志 + 英灵间博弈（F5-F8）

**F5 · 他不等你的指令**（主 A · 第一人称）
- **画面**：第一人称：英灵的背影跃上窗台，披风扬起，夜色城市在他身前铺开；他回头丢下一句话的神态
- **配文**：*"本王去会会这座城里的对手。你且歇着。"——他不需要你的许可。英灵从被召唤那一刻起，就有自己的意志。*
- **原作锚点**：Iskandar 式擅自行动（自行决定夜袭与赴会，事后通报）；英灵夜间自主活动设定
- **约束**：Wow 2 开门拍；你的反应克制（愣在原地即可）

**F6 · 无人指挥的遭遇战**（战场视角 · 全片唯一非第一人称帧）
- **画面**：桥下仓库区的深夜：两位英灵交锋定格——王者挥剑大开大合，枪兵低伏游走，兵刃相接的火花照亮两张神情迥异的脸。四周空无一人：**没有任何 Master 在场**
- **配文**：*此刻，没有任何玩家在线。两位英灵在各自的判断下相遇、试探、交手——互相隐藏真名，点到即止。*
- **原作锚点**：F/SN・F/Z 的"第一夜遭遇战"（Saber vs Lancer）——试探虚实、隐藏真名与宝具（M2 信息战）、自主收手；职阶战法差异（S4）
- **约束**：落地为视频/实机时此帧可处理为"战报回放"；英灵间博弈由明确机制承载（侦察 → 遭遇 → 职阶试探 → 真名隐藏 → 撤退判断），不暗示 AI 自由涌现斗智（Refuted 层护栏）

**F7 · 战报**（主 A · 第一人称）
- **画面**：第一人称低头看手机，英灵的战报消息气泡，背景虚焦的房间与窗外夜色。画面内文字：*"小子，遇上个有趣的枪兵。真名没露，我的也没有。他的 Master 快到了——你也过来，我给你留了位置。"*
- **配文**：*你是从他的战报里，才知道战争已经开始。他替你判断，也给你留了一个只有 Master 能补的位置。*
- **原作锚点**：M1 活人关系——Servant 主动通报战况；消息只做约定与衔接（继承旧红线，规避"聊天游戏"观感）
- **约束**：消息短、信息密度高、不闲聊

**F8 · 另一侧的战报**（副 B · 第一人称 · 冷调标识）
- **画面**：冷调 + 黑边。玩家 B 快步下楼、边走边看手机，他的英灵的战报是军用简报体。画面内文字：*"目标：王者级从者。位置：桥北仓库区。其 Master 未到场。请求指挥。"*
- **配文**：*同一场遭遇的另一侧——对面的玩家也在赶来。两对搭档，两种完全不同的语言。*
- **原作锚点**：英灵差异化通过沟通风格与行为模式传达，不靠解说
- **约束**：与 F7 构成"同一事件、两个玩家、两份第一人称"的对照——多人真实感的第一锚点

### Wow 3 · Master 配合互动（F9-F13）

**F9 · 到场 · 协商**（主 A · 第一人称）
- **画面**：第一人称抵达战场边缘：英灵侧身贴近，一手按住你的肩、一手指向前方的敌影（枪兵与其 Master 的两个剪影），眼神在夜色里发亮
- **配文**：*他出方案："枪兵的眼睛只会盯着我。你从暗处走到桥灯下——只要三秒，他的枪尖会晃那么一下。剩下的交给本王。"你承担的，是只有 Master 能补的那一环。*
- **原作锚点**：战术协商与分工（"你引开，我来打"式角色互补）；Waver 被拉上牛车并肩——Master 被英灵**需要**；G3 谈判 = 调研判定的最强 AI 展示面
- **约束**：你不是发号施令者也不是观众；台词短促、行动导向；**AI 标签唯一显性露出点**（自然语言实时协商，4 条件 gate 全过）

**F10 · 对面的配合**（副 B · 第一人称 · 冷调标识）
- **画面**：冷调 + 黑边。B 的第一人称：他半蹲在掩体后对通讯低声下令，视野前方枪兵型英灵贴墙潜行占位、以手势回应确认
- **配文**：*对面也在配合——他指挥，英灵执行。同样默契，路数完全相反。这是两对搭档的对决，不是打 NPC。*
- **原作锚点**：M1 羁绊的多样性——每对 Master-Servant 的相处方式独一无二
- **约束**：与 F9 同构镜像，构图上形成对照

**F11 · 决胜 · 半拍之差**（主 A · 第一人称）
- **画面**：第一人称：你从暗处踏入桥灯光圈的瞬间——视野左侧枪兵的枪尖偏移一瞬，正前方你的英灵已欺近到枪兵身前，剑光落下的前一帧。时间凝固感
- **配文**：*胜负不在数值，在配合——你们比对面，快了半拍。*
- **原作锚点**：S3 战斗爽感的"因果反转"节奏；配合决定胜负 = tagline 的直接兑现
- **约束**：你的动作是走位与暴露自己（战术执行），Master 不越位输出；保持第一人称

**F12 · 败者也是真人**（副 B · 第一人称 · 冷调标识）
- **画面**：冷调 + 黑边。B 的第一人称：他的枪兵负伤单膝撑地、反手把长枪横在 B 身前护主，侧脸低声说了什么；B 攥紧的手在画面边缘。远处，你与你的英灵的剪影并立
- **配文**：*输掉的一方，也是一位真人玩家和他的搭档。枪兵护着他的 Master 退入夜色——明晚，他们还会回来。*
- **原作锚点**：英灵护主（契约的双向义务）；败而不亡 = 战争持续感
- **约束**：败方保持尊严，不做挫败妖魔化；可选远处另一簇火光暗示战争多方持续，但不做数量话术

**F13 · 认可**（主 A · 第一人称）
- **画面**：战后桥头，晨光初起。英灵回身面向你，俯视镜头微笑，一只手掌粗暴地按住你的头顶（Iskandar 式亲昵）。背景城市天际线苏醒
- **配文**：*"今夜起，你是本王认可的 Master。"——召唤英灵，建立羁绊，找到只属于你们的配合方式，赢下圣杯。这场战争与现实同步，此刻，正在进行。*
- **原作锚点**：M1 认可时刻（Iskandar 对 Waver 的认可弧线）；tagline 收束
- **约束**：认可克制不煽情；tagline 全篇仅此一次

## 4. 证据与约束对照

**3 个必现证据时刻的兑现**（project-background 设计规则 1）：

| 必现证据 | 兑现帧 |
|---|---|
| ① 个人种子 → 世界/角色在眼前生成 | F1-F2（圣遗物 → 显形）+ F4（不同圣遗物 → 不同英灵，双重曝光） |
| ② AI agent 对无法预演的输入作出活反应 | F5-F6（无主自主博弈）+ F9（实时战术协商） |
| ③ 不止这一个世界/玩家 | F4 / F8 / F10 / F12（副线四帧：同一战争的另一位真人玩家） |

**AI 必要性 4 条件 gate**：AI 标签只在 F9（自然语言协商，四条全过）与 F2（圣遗物生成，旧 handoff §9 已验证全过）语境下成立；F6 的英灵间博弈属"单次说服性演示"范畴（G4 单次可过，长期多局不承诺），落地演示前需机制托底验证。

**继续有效的红线自查**：

| 红线 | 本文档兑现 |
|---|---|
| 不通过独白立人设 | F3 地球仪动作 / F7・F8 战报措辞风格 / F6 战法差异 |
| 不演玩家学习挫败 | 主线无挫败段；F12 败方是对手且保持尊严 |
| 离线消息只做约定衔接 | F7・F8 各一条消息，短促、战况通信性质 |
| 回避数量话术 | 全篇只出现 2 对；F12 火光暗示为可选且克制 |
| 玩家反应克制 | F5・F11・F13 约束注明 |

## 5. 待定项与下游工作

1. **对面败因（F11-F12 之间的隐含逻辑）**：主案 = 中性败因（职阶克制 + 配合执行差半拍）；备案 = 令咒强制致败（对面 Master 慌乱中用令咒强令英灵，配合断裂——原作"切嗣式强制 vs Waver 式信任"主题的玩法化，但有把令咒妖魔化的风险，需谨慎处理令咒首次亮相的语义）
2. **帧数弹性**：核心 13 帧；可压缩至 10（F5 并入 F7、删 F10）；可扩展至 15（增加令咒存在感帧、圣遗物来历闪回帧）
3. **示范英灵最终确认**：X 默认 Iskandar；Y 为不锁定具体原作角色的枪兵型示范，是否改用原创英灵（衔接旧 Wow 3 的"AI 生成新角色"证据）待定
4. **图片制作规格**：每帧的画幅比例、分辨率、生成工具（AI 生成 / 手绘概念图）、风格统一化方案交下游
5. **画面内文字的多语言版本**：战报消息文案目前为中文示范，demo 对外呈现语言待定
