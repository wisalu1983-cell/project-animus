# 动态骨架 × 具象且个性化的内容 — 论纲展示页

将 `docs/dynamic-skeleton-thesis.md` 的观点、逻辑与内容以可视化方式呈现的单页展示网站。

## 文件

| 文件 | 说明 |
|---|---|
| `index.html` | 页面结构，六个章节对应文档六章 |
| `style.css` | 「墨与烬」暗色编辑风样式，Noto Serif SC + Noto Sans SC |
| `script.js` | Hero 粒子关系网动画、滚动 reveal、导航高亮、阅读进度 |

## 使用

纯静态页面，无构建步骤。直接用浏览器打开 `index.html`，或本地起服务：

```bash
python -m http.server 8080
# 访问 http://localhost:8080
```

字体通过 Google Fonts 加载，离线时会回退到系统中文字体。

## 可视化对应关系

- 第 1 章三句话 → 三支柱卡片；「骨架」两层定义 → 双卡
- 第 2 章二选一局面 → 2×2 矩阵图（空白格高亮脉动）；模板 vs 开放输入 → 左右对比
- 第 3 章微观累积→宏观质变→反哺 → 循环流程图；三个局限 → 编号卡片
- 第 4 章分层生态 → 同心圆 SVG（玩家/生活/关系/影响范围）
- 第 5 章三个 Gap × 14 项能力 → 分组卡片，成熟度以彩色徽章标注，场景引文内嵌
- Hero 背景粒子关系网隐喻「活着的世界状态网络」，鼠标移动时节点会避让响应
