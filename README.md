# Project Animus

用 AI 实现《刺客信条》Animus 的核心体验：让任何人进入一段可交互的历史人生。

## 概念

Animus 在游戏设定中是"遗传记忆读取机"，但在 Ubisoft 的现实制作中，它是一套内容生产框架——选定人物、定义身份、嵌入历史、生成可探索的记忆序列。

Project Animus 的目标是用 AI Agent 自动化这套流程，把手工创作变成可个性化生成的历史沉浸系统。

## 核心能力映射

| Animus 设定能力 | 现实实现路径 | 可行性 |
|---|---|---|
| 历史场景重建 | 3D 重建 + 生成式世界模型 | 可实现 |
| 历史人物交互 | LLM Agent + AI NPC + 时代认知边界 | 可实现 |
| 沉浸式体验 | VR/AR + 空间音频 + 触觉反馈 | 可实现 |
| 同步率 | 证据置信度 + 历史偏离检测 | 可实现 |
| 记忆序列生成 | 多 Agent 协作 + 历史资料 RAG | 可实现 |
| DNA 读取祖先记忆 | — | 不可实现（科幻前提） |

关键洞察：**Animus 的体验问题是工程问题，Animus 的真实性问题是科学断点。绕开科学断点，直接解决体验问题。**
